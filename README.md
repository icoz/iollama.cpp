# iollama.cpp

Лёгкий HTTP-сервер на Go, предоставляющий OpenAI-совместимый API для LLM в формате GGUF через llama.cpp.

## Особенности

- Zero CGO (purego)
- Единый бинарник
- Автоматическая загрузка моделей с Hugging Face
- Поддержка потоковой передачи (SSE)
